Testing: JS Invocation...

Here's the function that grooms these <code> blocks...

/* * Run highlight() on every <code> block on the page. */ function highlight_all () { var i, n, e = document.getElementsByTagName('code'); for (i=0; i<e.length; i++) { /* * Split language and scope level. * Apply highlighting... */ n = e[i].getAttribute("lang").split(':'); highlight( e[i], n[0].toUpperCase(), n[1]?n[1]:0 ); /* * If there was a :N level, it must be cropped * from the lang attribute so it will be * recognized in the stylesheet as belonging * to the base language. */ if (n[1]) { e[i].setAttribute('lang', n[0]); } } }

Different usages of highlight () may vary. The example above is in the source of this page and is only one usage.
